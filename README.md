Fantasy Sports App (Flutter)

Short Description:
A fantasy sports mobile application built with Flutter that allows users to create teams, join contests, track live scores, and win prizes based on real player performances. Integrated with Razorpay for payments, local storage, and state management using Provider and Bloc.
********************
Project Overview
11 Squad is a fantasy sports platform where users can create virtual teams, join leagues, and compete with others. The app delivers a seamless experience with real-time score updates, smooth performance, and a secure payment flow.

As part of the development team, I contributed to:
<ul>
  <li>Building user-friendly UI with Flutter.</li>
  <li>Integrating payment gateway for contest entry fees using Razorpay.</li>
  <li>Implementing state management using Provider and Bloc.</li>
  <li>Enhancing app performance and scalability.</li>
  <li>Collaborating with cross-functional teams to ensure a robust and engaging user experience.</li>
</ul>

***********************
Key Features
<ul>
  <li>Team Creation: Pick players from real-life sports teams to form your fantasy squad.</li>
  <li>Contests & Competitions: Join leagues and compete with other users to win prizes.</li>
  <li>Live Score Updates: Stay updated with real-time match scores and track your team’s performance.</li>
  <li>Payment Integration: Secure in-app payments via Razorpay.</li>
  <li>User-Friendly Interface: Simple and engaging UI for smooth navigation.</li>
  <li>Performance Optimization: Optimized for high-traffic match hours and large-scale contests.</li>
</ul>

*******************************
Tech Stack
<ul>
  <li>Framework: Flutter (Android focus)</li>
  <li>State Management: Provider, Flutter Bloc</li>
  <li>Database: SQFlite (local storage), SharedPreferences (lightweight storage)</li>
  <li>Networking: Dio (API calls)</li>
  <li>Payments: Razorpay Flutter SDK</li>
  <li>Architecture: MVVM + Bloc pattern</li>
</ul>
